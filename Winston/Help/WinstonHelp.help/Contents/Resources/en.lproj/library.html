Building your library

Importing books

  • Drag files onto the window, use the Add Files toolbar button, or press ⌘O.
  • Supported formats: EPUB, MOBI, AZW3, PDF, TXT, HTML.
  • Imported files are copied into Winston's library folder, so moving or deleting the originals won't affect your library.
  • To import automatically, turn on a watch folder in Settings → Auto‑Import. New books dropped there are added for you (duplicates are skipped).

Importing from calibre

  • Choose File → Import from Calibre Library… (⌘⇧O) and pick your calibre library folder (the one containing metadata.db).
  • Winston reads calibre's database directly — calibre does not need to be running or even installed — and brings each book in with its cover, author, series, tags, rating, publisher, year and description.
  • When a book has several formats, the Kindle‑friendly one is chosen (AZW3, then MOBI, AZW, EPUB, PDF, TXT).
  • The batch is grouped into a dated “Calibre Import” collection so you can review it, and books already in your library are skipped.

Views, search & sorting

  • Toggle between the grid and the list with the segmented control in the toolbar.
  • Sort by Title, Author, Date Added, or Rating from the sort menu; in the list, click a column header.
  • Search with the toolbar search field (⌘F). Plain text matches title, author, tags and series.
  • Field search: narrow with author:, tag:, series:, title:, format: and year: — for example author:tolkien tag:"science fiction" year:>2000. Use quotes for values with spaces and >/</= with years.
  • The sidebar filters by format, author, series, tag, recently added, and rating.
  • Smart collections: with a search active, choose Library → Save Search as Collection…. It appears in the sidebar with a ✦ icon and always shows the books currently matching that search.

Searching inside books

  • Choose Library → Search Inside Books… or press ⌘⇧F.
  • Winston searches text in EPUB, PDF, TXT, and HTML files and groups each result as book → chapter or page → excerpt. Click an excerpt to open the book, or use Show in Library to select it in the normal library view.
  • The index stays on this Mac and makes no network requests. Refreshing it reuses every book whose content hash is unchanged; text is extracted again only after the book file changes.
  • Image-only scanned PDFs have no extractable text and cannot be searched without OCR.

Choosing what to read today

  • Choose Library → What Should I Read Today… or press ⌘⌥S.
  • Choose whether to continue or start fresh, a preferred length, a mood or genre tag, language, and whether to continue a series. Ratings and time spent waiting in your library can act as tie-breakers.
  • Winston considers unread, reading, and paused books whose primary file is available. For numbered series it will not recommend a later unread volume ahead of an earlier one.
  • Every pick explains its strongest reasons. Use Try Another to move through the remaining matches, Open in Reader to read it, or Show in Library to select it in the normal view.
  • Recommendations use only metadata already stored in your local library and make no network requests.

Collections & reading status

  • Create a collection from the + next to “Collections” in the sidebar; rename or delete via its right‑click menu.
  • Add books to a collection with the book's right‑click menu → Add to Collection.
  • Books saved from Discover appear in the built-in Wishlist smart collection. Importing the same title and author removes it from the wishlist and confirms that it is now in your library.
  • Set reading status (Unread / Reading / Finished) from the inspector or the right‑click menu. The sidebar can filter by Reading and Finished.

Importing reading history

  • Choose File → Import Reading History… and select a CSV export from Goodreads, StoryGraph, or Hardcover.
  • Winston matches local books by ISBN first, then by title and author. Title-only and ambiguous matches wait for your review; unmatched rows are skipped.
  • The preview shows the exact status, rating, dates, and reading cycles that will be added. Reimporting the same file does not duplicate existing cycles, and existing cycles are never deleted.
  • The CSV is processed entirely on this Mac. Importing reading history makes no network request and does not create missing book files or catalog entries.

Editing metadata & covers

  • Edit one book from the inspector's Edit button, or many at once by selecting them and choosing Edit Metadata for N….
  • Rename an author, series, or tag across the whole library from its right‑click menu in the sidebar.
  • If an author was imported reversed (e.g. “Tolkien, J.R.R.”), a tip at the bottom of the sidebar offers to fix it to “J.R.R. Tolkien” in one click.
  • Change a cover by dropping an image onto it in the inspector, or right‑click the cover → Choose Cover… / Reset Cover.
  • Find and clean up copies with Library → Find Duplicates… — copies are matched by title and author (regardless of name order), and each group has a Keep action that removes the others.

Highlights

  • Winston collects the highlights and notes you make on a Kindle. Import them from the device view (see Sending & syncing); a book with highlights shows a 💬 badge.
  • Open Library → Highlights… to browse every highlight across your library, grouped by book, and search their text.
  • Click Export… to save them as Markdown — one “Author - Title.md” file per book — ready for Obsidian, notes apps, or backup.

Fetching metadata & community ratings online

  • Turn on Settings → Metadata → Fetch metadata online. Winston then looks up covers and details from free catalogs (Open Library, Google Books) for imported books.
  • Only missing fields are filled — your existing or edited data is never overwritten — and if nothing is found, the book keeps its embedded metadata.
  • You can also fetch on demand: select books and choose Book → Fetch Metadata Online.
  • Community ratings (how other readers scored a book, shown beneath your own rating) come from Hardcover when you add a free token in Settings → Community Ratings, otherwise from Google Books / Open Library. Each book is looked up once; adding a token later? Re‑fetch with Fetch Metadata Online.
  • Open Library → Series… with a Hardcover token configured to compare each local series with its main catalog: Winston shows how many volumes you own, names the missing books, and links directly to each title on Hardcover.
  • When the setting is off, Winston makes no network calls and works fully offline.

Backup & library health

  • Turn on Settings → Backup and choose a folder. Winston then copies your catalog and covers there about once a day, keeping the most recent few backups.
  • Choose Browse Backup History… to compare any snapshot with the current library. You can restore only a book's metadata, only its cover, or all of its catalog data. Winston creates a new safety backup before every restore.
  • Time Machine never replaces EPUB or PDF files. If you restore a catalog entry whose book file is gone, Winston keeps the restored record and asks you to relink the file.
  • This is a safety net for your organization (ratings, collections, reading status, highlights). To move the actual book files somewhere portable, use Library → Export Library…, which writes readable “Author - Title” files plus a metadata CSV/JSON.
  • If a book's file is moved or deleted outside Winston, the book is flagged with a ⚠️ badge; right‑click it and choose Replace File… to point it at the file again.
